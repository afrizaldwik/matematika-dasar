## ğŸ¨ Kustomisasi

### Mengubah Warna Tema
Warna gradasi default:
```css
background: linear-gradient(135deg, #40bf8c 0%, #bbe4d0 50%, #eeeeee 80%, #dddddd 100%);
```

Untuk mengubah warna, edit variabel CSS di bagian `:root`:
```css
:root {
  --primary-color: #40bf8c;
  --secondary-color: #bbe4d0;
  --light-color: #eeeeee;
  --neutral-color: #dddddd;
}
```
